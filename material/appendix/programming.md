## コンパイル

## 型

- `int` : 整数
- `float` : 小数
- `boolean` : 真偽値
- `String` : 文字列

## 等号・不等号

- `==` : 等しい
- `!=` : 等しくない

## 変数

- 変数の宣言

```java:
int x;
x = 10;
println(x);
```

- 変数を宣言するときに初期化する

```java:
int x = 10;
println(x);
```

- 変数の代入（更新）

```java:
int x = 7;
println(x);

x = 11;
println(x);
```

- 違う型の変数を宣言

```java:
boolean x = true;
println(x);

x = false;
println(x);
```

- 型が違う変数には値を代入できない

```java:
int x = 7;
x = 3.0; // エラーになる
println(x);
```
